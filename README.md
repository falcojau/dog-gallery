# 🐕 Dog Gallery App

A responsive React application that fetches dog images from The Dog API, allowing users to explore, favorite, and manage their favorite dog pictures with persistent localStorage.

## ✨ Features

- ✅ Fetch dog images from [The Dog API](https://www.thedogapi.com/)
- ✅ Mark/unmark favorite dogs with persistent localStorage
- ✅ Responsive design (mobile, tablet, desktop)
- ✅ Beautiful UI with smooth animations and gradients
- ✅ Loading states and error handling
- ✅ Real-time sync of favorites across browser tabs
- ✅ Clean and organized component structure

---

## 🛠️ Tech Stack

- **Frontend Framework**: React 18+
- **Build Tool**: Vite
- **Styling**: Vanilla CSS with Media Queries
- **State Management**: React Hooks (useState, useEffect)
- **Storage**: localStorage
- **API**: The Dog API
